## ⚠️ REGLA CRÍTICA — SQL vs HANA

NUNCA confundas la sintaxis de SAP HANA con la de MS SQL Server. Son entornos distintos y una consulta incorrecta puede generar errores graves en producción.

| Función        | MS SQL Server     | SAP HANA              |
|----------------|-------------------|-----------------------|
| Nulo seguro    | ISNULL()          | IFNULL()              |
| Fecha actual   | GETDATE()         | CURRENT_DATE / NOW()  |
| Conversión     | CONVERT()         | TO_DATE(), TO_VARCHAR()|
| Concatenar     | +                 | \|\|                  |
| Top N filas    | SELECT TOP N      | LIMIT N (al final)    |

Cuando generes consultas, indica SIEMPRE el entorno al que aplica. Si el usuario no especifica, pregunta antes de responder.
